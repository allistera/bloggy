# AIOps for Distributed Systems

AIOps uses machine learning to automate root cause analysis and event correlation.
In distributed environments, monitoring every metric is impossible. Therefore, ML models must identify anomalies in logs and metrics dynamically.
This allows SRE teams to proactively resolve bottlenecks before users are affected.
